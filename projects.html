Personal Projects

Squid Battle

"Squid battle" is an independent video game project developed in Unity. It is a local multiplayer party game for 2-6 players. Each player controls a squid character, and plays with the goal of pushing other players out of the battle arena to become the last player standing. The game is still in early development, but is planned for commercial release in the future.

PICO-8 Game Engine Development

PICO-8 is an 8-bit "fantasy console" designed for making small games and other programs in Lua. I am currently working alongside a friend to develop a game engine within PICO-8 optimized for it's unique software limitations. So far the engine features an object-oriented heirarchy, hitboxes, basic physics interactions (movement, collision, knockback), dynamic particle system, and AI navigation using A* algorithm. I look forward to developing a game that takes advantage of all the features in our engine.
